## uwsgi_ini_make

ruby が必要なのでrbenvかなにかをinstallして
ruby 2系をinstall してください

-

### usage

```
bundle install --path vendor/bundle
bundle exec ./uwsgi_ini_make make -r redbule -d redbule
```
